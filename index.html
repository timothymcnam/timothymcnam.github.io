Projects

Hack Hamp April 2017

At this event, I worked alongside other programmers, with the use of GitHub, to build a song generator. The overall product has a GUI to input a song name, which is then used to scan a text file for related lyrics. These lines are then used to build a markov chain, capable of generating unique song lyrics related to the title of your song! With the use of a sorting algorithm, lines in the song are then rearranged to rhyme. O(nlogn) time, in case you were wondering. Then the final song is returned to the GUI for any user to enjoy. Currently, we have the program set to generate rap songs; however, by editing the text file, it is possible to generate any type of song! The possibilities are endless.
